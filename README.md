# 奶龙桌宠 (Nailong Pet)

一个基于 C++ 和 SFML 开发的 Windows 桌面宠物程序，以网络热门形象"奶龙"为主角，支持丰富的交互功能和系统感知能力。

## 功能特性

- **点击互动**：单击奶龙触发微笑动画，快速连点5次触发大笑动画并播放音效
- **物理甩动**：拖动奶龙快速松手可将其甩出，支持重力、空气阻力、边缘弹跳和碰撞形变动画
- **系统监控**：实时读取 CPU 和内存使用率，过载时奶龙切换为哭泣状态
- **疲惫提醒**：连续使用电脑超过1小时后奶龙进入疲惫状态并提示休息
- **打招呼**：程序启动或电脑唤醒时奶龙显示问候气泡，包含时间、日期、IP属地和陪伴时长
- **陪伴时长**：累计记录奶龙陪伴时间，跨会话持久保存
- **右键菜单**：支持关闭、最小化到系统托盘、固定位置、禁用大笑、静音、刷新状态、查看系统信息

## 运行环境

- Windows 10 / 11
- 无需安装任何依赖，解压即用

## 使用方法

1. 下载 Release 页面的 `nailong-pet-release.zip`
2. 解压到任意文件夹
3. 双击 `nailong-pet.exe` 运行
4. 右键奶龙打开菜单，按 `Escape` 或右键选择关闭退出

## 编译方法

依赖：CMake、SFML 3.x、Visual Studio 2022

​```bash
mkdir build
cd build
cmake ..
cmake --build .
​```

## 技术栈

- **语言**：C++17
- **图形库**：SFML 3.x
- **平台 API**：Windows API（系统监控、托盘、透明窗口、网络请求）
- **构建工具**：CMake
